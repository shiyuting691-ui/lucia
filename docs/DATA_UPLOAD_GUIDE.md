# 资料上传指南 · DATA_UPLOAD_GUIDE

> 本文档说明如何通过看板「资料上传中心」页面上传和管理各类资料。  
> 全程不需要手动移动文件或敲命令。

---

## 一、什么资料上传到哪里

| 资料类型 | 上传位置 | 保存路径 |
|---------|---------|---------|
| 产品说明文档 | 资料上传中心 → 产品资料 | knowledge_base/01_产品知识库/ |
| 销售话术文档 | 资料上传中心 → 销售话术 | knowledge_base/02_销售话术库/ |
| 风控规则文档 | 资料上传中心 → 风控规则 | knowledge_base/06_风控表达库/ |
| 订单数据 | 资料上传中心 → 数据上传 | data/orders.csv |
| 咨询数据 | 资料上传中心 → 数据上传 | data/leads.csv |

## 二、上传中心怎么用

### 第一步：上传知识库文件

1. 打开控制台，点击侧边栏「📁 资料上传中心」
2. 进入「📚 知识库上传」标签页
3. 选择文件类型（产品资料 / 销售话术 / 风控规则）
4. 选择子类型（如：Final精准押题 / 价格异议 / 押题产品风控规则）
5. 上传文件（支持 .md .txt .docx .pdf）
6. 确认文件名后点击「💾 保存」按钮
7. 重复以上步骤上传更多文件

### 第二步：扫描登记文件

上传完所有文件后：

1. 切换到「⚡ 操作按钮」标签页
2. 点击「🔍 扫描知识库资料」按钮
3. 等待几秒，系统自动扫描所有文件并登记到数据库
4. 底部状态面板会显示已登记文件数量

> 扫描只登记文件信息（路径、名称、大小），不读取全文，不调用 Claude。

### 第三步：（可选）生成摘要

1. 点击「🤖 生成/更新资料摘要」按钮
2. 系统会对每份文档生成 150-300 字摘要 + 关键词
3. Agent 生成内容时优先读取摘要，更快更省 token

> 注意：此步骤调用 Claude API，消耗 token。建议文档更新后才操作。

### 第四步：上传和导入数据

1. 切换到「📊 数据上传」标签页
2. 上传 orders.csv（订单数据）并点击「💾 保存」
3. 上传 leads.csv（咨询数据）并点击「💾 保存」
4. 切换回「⚡ 操作按钮」标签页
5. 点击「📥 导入订单数据」
6. 点击「📥 导入咨询数据」
7. 点击「📡 更新市场信号」

## 三、CSV 字段说明

### orders.csv（订单数据）

| 字段 | 说明 | 示例 | 必填 |
|------|------|------|------|
| order_date | 下单日期 YYYY-MM-DD | 2026-06-01 | ✅ |
| school | 学校名称 | UCL | ✅ |
| country | 国家 UK/Australia | UK | ✅ |
| product | 产品ID | dissertation / final_prediction | ✅ |
| course_code | 课程代码 | ECON3001 | ❌ |
| deadline | 截止日期 | 2026-06-25 | ✅ |
| amount | 金额（人民币）| 9000 | ✅ |
| sales_owner | 负责销售 | Lucia | ✅ |
| status | 状态 completed/in_progress/cancelled | completed | ✅ |

### leads.csv（咨询数据）

| 字段 | 说明 | 示例 | 必填 |
|------|------|------|------|
| inquiry_date | 询盘日期 | 2026-06-01 | ✅ |
| school | 学校名称 | UCL | ✅ |
| country | 国家 | UK | ✅ |
| product_interest | 感兴趣产品 | dissertation | ✅ |

## 四、常见错误

| 错误 | 原因 | 解决方法 |
|------|------|---------|
| 导入后显示0条 | CSV 字段名拼写错误 | 下载模板对照检查字段名 |
| 文件保存后不显示 | 浏览器缓存 | 点击「🔄 刷新」或 F5 |
| 扫描后登记数量不变 | 文件格式不支持 | 检查文件扩展名是否为 .md/.txt/.docx/.pdf |
| 市场信号为空 | orders/leads 表无数据 | 先导入订单和咨询数据 |
| 摘要生成失败 | API 额度不足或文件损坏 | 检查 API Key，重试 |

## 五、如何确认看板已更新

1. **资料已登记**：资料上传中心底部「已登记知识库」数量增加
2. **订单已导入**：底部「订单记录」数量增加
3. **咨询已导入**：底部「咨询记录」数量增加
4. **市场信号已更新**：底部「市场信号」数量增加；前往「📡 市场情报台」查看新信号
5. **摘要已生成**：底部「已生成摘要」数量增加
